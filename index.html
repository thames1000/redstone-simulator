⚡ Redstone 3D tick 0
Controls
• Left-click: place selected block
• Right-click / X: remove block
• Drag: orbit   Scroll: zoom
• R: rotate hovered component
• Interact tool: toggle levers, press buttons,
  repeater = delay, comparator = mode, crop = grow
• Pistons push when powered; sticky ones pull back.
  Slime drags attached blocks along.
• Space: run/pause   . : single step
